# Garden & Orchard Inventory

Living inventory of what grows on the property. Tracks harvest windows by year, building a multi-year picture of when things produce.

**Currently available:** Items where the current year's "Last harvest" is blank.

---

## How This File Works

**Adding an item:** Create a section with a harvest log table. If it's currently producing, leave "Last harvest" blank for the current year.

**End of season:** Fill in the last harvest date. The item stays in place as a permanent record — do not delete it. Next year, add a new row.

**New year:** Add a new row to the harvest log with the new year. Leave "Last harvest" blank again when the season starts.

**Meal design:** Check this file first when designing ingredient-driven meals. If an item has no "Last harvest" for the current year, it's in season. Cross-reference remaining macro budget with the diet tracker if active.

**Pattern recognition:** Over years the log shows seasonal patterns: "Tomatoes usually start mid-June, peak July–August, done by October." Use this to plan ahead and avoid suggesting out-of-season produce.

**Cross-reference:** Link each item to recipes that use it. When designing a meal from available ingredients, prioritize combinations that appear in existing tracked recipes.
