# Coach's Notes Log

Reverse-chronological log of coach's notes rendered on the fancy dashboard. One entry per dashboard render. Purpose: maintain consistency across days so advice doesn't contradict itself or chase one-day noise.

**Format:** Each entry is a date header + the exact notes rendered, plus a `Context:` line noting what data drove the advice.

**Retention:** Keep 30 days rolling. Archive older entries to `archive/` when the file exceeds 200 lines.
